Project Idea

Overview

My project idea is to use IT to increase the amount of solar radiation that can be generated by a solar panel array. This can be used either as a business capability, by being a selling point due to the fact that the solar panel array will generate more electricity than the competition, or, , it can be used by a homeowner wanting to get the most out of their existing solar panels. Essentially it will involve using a small computer which will decide where the best part of the sky is to point the solar array for maximum sun exposure, which will in turn generate the most electricity it possibly can.

Motivation

My motivation for this project is due to the fact that I consider renewable energy production to be important and something that we as a society should be striving for. The move away from fossil fuels and toward renewable energy needs to be hastened if we are going to avoid a climate change disaster. We need to do the best that we can to harness as much solar energy as possible, and we can start by using solar tracking solar arrays. According to a study by Hafez et al. (2018) using a solar tracking system can increase solar generation by 8-50%, based on many factors such as weather, location and tracking type to name just a few.

Description

This project has a one main system which controls and drives a number of subsystems. The main system will be a small form factor computer which will be responsible for doing all the computing for the two subsystems. It will use a Raspberry Pi to do all the processing required and handle hosting features, as well as use inputs from wind and light sensors as well as GPS receiver.

The first subsystem, which is by far the most important part, is the solar tracking system. The system will be able to do this by itself using its various inputs. In this subsystem there will be two main components; the first relates to tracking the Sun’s path. It will do this initially by using the unit’s GPS position and then inputting that value into the solar positioning algorithm. By using this method it will give an approximate location of where the Sun will be located in relation to the unit and drive the solar array into position. Once in position, the unit will then do fine-tuning by using an onboard light sensor, which will input values from the UV and infrared spectrum. The computer will use these values to fine tune the array for maximum energy harvesting.

The second component relates to driving the solar array into position. It will do this using the data from the first component to drive various electric motors for linear actuators combined with a motor controller to move the array into position. There are two options available for the array: using either a single axis or a dual axis tracking array. Research(1) has shown that in the Australian environment, using a single axis array will potentially increase your solar harvesting capability by roughly 20%. However, using a dual axis array will potentially provide an additional 5-10%. Having two options available enables customers to pick which system is best for them.

The second component of the main system is the website hosting services that the Raspberry Pi will provide. The website will provide the system owner with the ability to control and monitor their solar array. It will hopefully provide a plug into their solar inverter and batteries (if they have them) to display their current kWh generation as well as the battery charge levels and total power produced versus actual power drawn.

Once the array is built, the system will only need a small amount of time to set up, which simply involves connecting it to an internet source via wifi, cellular or cabled connection if available, and then an initial configuration. Once done, the system will be “set and forget” once it is up and running, and will require next to no maintenance save for the solar array tracking mechanics.

Future works will hopefully be to fully integrate all aspects of your home/business solar system into the website, and to incorporate a mobile application for ease of use wherever you are in the world.

Skills Required

To get this project up and running will require a range of skill sets. The three main skills will be:

  1. Website Developer - This person will be responsible for creating and managing the website using primarily Java which will integrate with the solar tracking software. They will also be responsible for the setup and maintaining the Apache web server.
  2. Program Developer - Will create the solar tracking software using Python, which will take input from the various sensors to figure out where to drive the array to get maximum exposure from the sun.
  3. Electrical Engineer - This person will be responsible for taking the outputs of the solar tracking program and turning them into electrical instructions to drive the motors/linear actuators.

These three core positions will be easily fill-able as they only require basic level skills, as the requirements are not too complex. Furthermore, the hardware is commercially and readily available which will help the project as nothing is design specific.
